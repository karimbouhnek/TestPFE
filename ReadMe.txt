Application Web développé a l'aide de NodeJS.

Instruction pour lance l'application:
	1.Télécharger et installer mongodb a partir de ce lien https://www.mongodb.com/download-center/community
	2.Décompresser le projet dans un dossier.
	3.Ouvrir le terminal.
	4.Naviguer vers le dossier extrait.
	5.Taper npm install et appuyer sur entré.
	6.Taper nodemon app et appuyer sur entré.
	7.Utiliser le navigateur pour accéder a l’URL http://localhost:5000.
